# HideChat - WhatsApp Privacy Extension

**HideChat or Blur WhatsApp Messages** is a lightweight, open-source browser extension designed for WhatsApp Web users who prioritize privacy. It allows you to blur sensitive information like contact names, message previews, and chat content, ensuring your private conversations remain hidden from prying eyes.

## ✨ Key Features
- **Sidebar Blur:** Automatically blurs contact names and message previews in your chat list.
- **Message Panel Blur:** Keeps your active chat messages blurred for maximum privacy.
- **Toggle Controls:** Use the extension popup to enable or disable blur effects for the sidebar and messages independently.
- **Hover-to-Reveal:** Need to check a message? Simply hover over the blurred area, and it will become clear instantly.
- **Always-Clear Profile:** Profile pictures and group logos remain visible, so you can easily identify chats without compromising privacy.
- **Performance Optimized:** Built with minimal CSS and JavaScript to ensure zero lag or performance impact on WhatsApp Web.

## 🚀 How to Install
1. **Clone or Download:** Click the "Code" button above and select "Download ZIP" or clone this repository.
2. **Open Extensions:** In your browser (Chrome/Edge/Brave/Opera), navigate to `chrome://extensions/`.
3. **Enable Developer Mode:** Toggle the **"Developer mode"** switch in the top-right corner.
4. **Load Unpacked:** Click the **"Load unpacked"** button and select the folder where you saved this project.
5. **Use:** Open [WhatsApp Web](https://web.whatsapp.com), and use the extension popup (click the puzzle icon in your browser toolbar) to toggle your privacy settings.

## 🔒 Privacy & Security
This extension is designed with **Privacy First**.
- **No Data Collection:** This extension works strictly locally in your browser. 
- **No Third-Party APIs:** It does not collect, store, or transmit any of your personal messages or data to any external server. 

## 🛠 Tech Stack
- **JavaScript (Vanilla):** For managing toggle states and user preferences.
- **CSS3:** For high-performance blur filters and smooth transitions.
- **Chrome Storage API:** To save and sync your settings across browser sessions.
